# Python Projects Collection

This repository contains a collection of Python projects developed through coursework and personal practice. The goal of this repository is to showcase different programming concepts, problem-solving approaches, and project implementations using Python.

One of the main projects included is a Python-based text adventure game inspired by Ragnarok Online. In this game, the player navigates through different locations, collects items, and prepares for a final objective. The repository also includes other projects, such as a Higher and Lower guessing game, along with additional programs that demonstrate core programming skills.

This repository is actively being expanded, and more Python projects will be added over time.

## Projects Included

### Ragnarok Text Adventure Game

A Python-based text adventure game where the player explores multiple locations, collects items, and completes an objective before entering the final area. The game uses a dictionary-based structure to manage room navigation and tracks player inventory throughout the experience.

### Higher and Lower Game

A number guessing game where the user defines a lower and upper bound, and the program generates a random number within that range. The player continues guessing until the correct number is found. This project demonstrates input validation, loops, and conditional logic.

## Skills Demonstrated

* Control flow (loops and conditionals)
* Input validation and user interaction
* Dictionary-based data structures
* Basic game design and logic implementation
* Problem-solving and program structure
